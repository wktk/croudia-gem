# Croudia

A Mechanize-based Ruby scraper for [Croudia](https://croudia.com)

## Installation

Add this line to your application's Gemfile:

    gem 'croudia'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install croudia
